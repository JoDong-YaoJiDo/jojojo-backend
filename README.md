# Local Community Backend

FastAPI + SQLAlchemy + SQLite 기반 지역 정보 공유 백엔드.

## 문서

- Swagger UI: `/docs`
- ReDoc: `/redoc`
- OpenAPI: `/openapi.json`
- ERD: `docs/erd.mmd`
- API 명세 초안: `docs/api.md`

## 실행

1. `.env` 생성
2. `tourism.json` 경로 설정
3. `pip install -r requirements.txt`
4. `uvicorn main:app --reload`
